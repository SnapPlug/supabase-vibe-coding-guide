# 04. 스키마 설계 (Schema)

테이블 구조, 관계, 인덱스 설계. AI는 빠르게 스키마를 만들어주지만 성능과 확장성을
고려하지 않은 설계를 내놓는 경우가 많다.

## 문제 목록

| # | 제목 | 심각도 | 상태 |
|---|------|--------|------|
| - | (문제 발견 시 추가) | - | - |

## 자주 발생하는 패턴

- 인덱스 없는 FK 컬럼으로 N+1 쿼리 폭탄
- `jsonb` 남용으로 쿼리 최적화 불가
- `updated_at` 자동 갱신 트리거 누락
- soft delete 패턴 없이 `DELETE` 사용
