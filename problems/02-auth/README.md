# 02. 인증 (Auth)

Supabase Auth는 편리하지만, 바이브코딩 환경에서 JWT 처리, 세션 갱신, OAuth 설정 등에서
반복적인 실수가 발생한다.

## 문제 목록

| # | 제목 | 심각도 | 상태 |
|---|------|--------|------|
| - | (문제 발견 시 추가) | - | - |

## 자주 발생하는 패턴

- JWT 만료 처리 누락 → 조용한 인증 실패
- `auth.uid()` vs `auth.role()` 혼동
- 클라이언트 vs 서버사이드 세션 처리 차이
- OAuth 리다이렉트 URL 설정 오류
