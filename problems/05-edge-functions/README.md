# 05. Edge Functions

Deno 기반 서버리스 함수. 로컬 개발 환경과 배포 환경의 차이, 의존성 관리,
시크릿 처리 등에서 반복적인 문제가 발생한다.

## 문제 목록

| # | 제목 | 심각도 | 상태 |
|---|------|--------|------|
| - | (문제 발견 시 추가) | - | - |

## 자주 발생하는 패턴

- Node.js 패키지를 Deno 환경에서 사용하려는 시도
- 환경변수 `Deno.env.get()` 누락
- CORS 헤더 설정 오류
- `service_role` 키를 Edge Function 내부에서 하드코딩
